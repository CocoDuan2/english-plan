## 学习者信息
- 年龄：3岁
- 基础：学过牛津自然拼读，会简单词汇
- 每日时间：约25分钟
- 开始日期：2025-02-25

## 整体规划思路

### 核心原则
- **每天一个主题**，但连续两天不重复同一主题
- **每天两套课件**：外教课件（teach.html）+ 课后复习游戏（review.html）
- **每天学3-5个新词** + 复习之前的词
- **主题轮转**：颜色→动物→身体→食物→数字→动作→...循环推进
- 同一主题隔几天再回来，螺旋式加深

### 课件 A：外教课件 teach.html（25分钟）
线上外教通过屏幕共享展示，外教控制节奏。
1. **Warm-up Review**（3分钟）— 闪卡快速复习旧词
2. **New Words**（8分钟）— 逐个引入新词，大图+单词，外教带读+造句
3. **Guided Practice**（6分钟）— TPR、问答、指认等互动
4. **Listen & Do**（5分钟）— 外教说指令，孩子做动作/指图
5. **Wrap-up**（3分钟）— 总结复习

### 课件 B：课后复习游戏 review.html（25分钟）
小孩自己操作，家长辅助。
1. **复习热身**（4分钟）— 用游戏回顾之前学过的词（跨主题）
2. **新词学习**（6分钟）— 今天主题的3-5个新词
3. **互动游戏1**（5分钟）— 围绕新词的互动（涂色/孵蛋/等）
4. **听声找图**（5分钟）— 家长读提示，孩子找（新词+复习词混合）
5. **综合游戏**（5分钟）— 记忆翻牌/打地鼠/分类等（新旧词混合）

### 句型（自然融入，不单独教）
- What's this? It's a ___.
- What color is it? It's ___.
- Touch your ___.
- I like ___.
- How many?
- Let's ___!

### 复习机制
- 每5天一个复习日（纯游戏，不教新词）
- 艾宾浩斯间隔：学后第1/2/4/7天复习
- 复习热身环节每天都会带入之前的词（跨主题）
- 家长反馈随时调整

### 互动游戏类型（轮换使用）
1. 颜料瓶涂色 — 点击灰色变彩色
2. 孵蛋 — 点击蛋孵出动物/物品
3. 听声音找图片 — 老师说，孩子点
4. 记忆翻牌 — 配对游戏
5. 打地鼠 — 目标物品冒出来，快速点击
6. 分类游戏 — 把物品拖到正确的篮子里
7. 连线 — 把词和图片连起来
8. 拼图 — 碎片组合
