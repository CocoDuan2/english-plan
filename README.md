# Ranran English Plan 🐷

## 学习者
- 3岁，有自然拼读启蒙
- 每日15-25分钟
- 喜欢佩琪

## 课件形态
- **teach.html** — 外教屏幕共享幻灯片，老师带着走
- **review.html** — 课后复习游戏，小孩自己玩

## 视觉风格
- 佩琪作为引导角色深度融入每页
- 颜色主题用 SVG，其他主题用 Twemoji CDN
- 佩琪风配色：粉色 #F9ADC7、天蓝 #7EC8E3、草绿 #8BC34A

## 课程设计理念（故事化学习）

### 核心原则
- **每课一个故事**：不是孤立的词汇表，而是佩琪的一天/一次冒险
- **情境化学习**：词汇出现在真实场景中（野餐、动物园、早晨起床...）
- **情感连接**：小孩能想象自己和佩琪一起做这件事
- **RAZ 递进式**：无专门复习日，旧词通过故事线自然复现

## 故事化教学要点

### teach.html 故事线设计（10页）
- **Title 页**：故事标题 + 场景预告
- **Hello 页**：佩琪邀请："Come with me!"，引入今天的冒险
- **新词页**（5页）：每个词都在故事情境中出现，RAZ句式重复
- **互动页**（2页）：每天不同的互动（听词+游戏）
- **Bye 页**：佩琪说再见，预告明天的冒险

### review.html 故事延续（5模块）
- **Start** → **Learn** → **Game** → **Memory** → **Done**
- 游戏设计要符合故事主题
- 例如 Day 4 Picnic：帮佩琪准备野餐篮
- 例如 Day 2 Zoo：帮佩琪找走丢的动物
- 无 Yesterday Review，专注今天的5个新词

### 情感连接点
- 使用第一人称："I see"、"I like"、"I can"
- 鼓励小孩说出自己的经历："你也去过动物园吗？"
- 每课结束时的成就感："You helped Peppa! Great job!"
