## 学习者信息
- 年龄：3岁
- 基础：学过牛津自然拼读，会简单词汇
- 每日时间：约25分钟
- 开始日期：2025-02-25

## 整体规划思路

### 核心原则
- **每天混合主题**，不要一周只学一个主题，避免无聊
- **每天25分钟**，分5个模块，节奏快、花样多
- **每天接触5个词**（2-3个新词 + 2-3个复习词），跨主题混合
- **螺旋式推进**：同一个词在不同场景反复出现

### 每日结构（25分钟 = 5个模块）
1. **复习热身**（4分钟）— 用游戏回顾之前学过的词
2. **新词学习A**（5分钟）— 主题A的2-3个新词（如颜色）
3. **互动游戏A**（5分钟）— 围绕新词的互动游戏
4. **新词学习B**（5分钟）— 主题B的2-3个新词（如动物），或复习旧词
5. **综合游戏**（6分钟）— 把今天所有新旧词混在一起的大游戏（记忆翻牌/寻宝等）

### 句型（自然融入，不单独教）
- What's this? It's a ___.
- What color is it? It's ___.
- Touch your ___.
- I like ___.
- How many?

### 自然拼读同步推进
- 每天复习1-2个已学字母音
- 配合当天词汇中的CVC词（cat, dog, red, pig等）
- 不单独开模块，融入学习过程

### 复习机制
- 每4天一个复习日（纯游戏，不教新词）
- 艾宾浩斯间隔：学后第1/2/4/7天复习
- 掌握度差的词自动加入下一天
- 家长反馈随时调整

### 互动游戏类型（轮换使用，保持新鲜）
1. 颜料瓶涂色 — 点击灰色变彩色
2. 孵蛋 — 点击蛋孵出动物/物品
3. 听声音找图片 — 老师说，孩子点
4. 记忆翻牌 — 配对游戏
5. 拼图 — 拖拽碎片组合
6. 打地鼠 — 目标物品冒出来，快速点击
7. 涂鸦画板 — 用手指画出指定物品
8. 分类游戏 — 把物品拖到正确的篮子里
