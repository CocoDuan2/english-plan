# Ranran English Plan 🐷

## 学习者
- 3岁，有自然拼读启蒙
- 每日15-25分钟
- 喜欢佩琪

## 课件形态
- **teach.html** — 外教屏幕共享幻灯片，老师带着走
- **review.html** — 课后复习游戏，小孩自己玩

## 视觉风格
- 佩琪作为引导角色深度融入每页
- 物品用 SVG 画图（颜色精确对应教学内容）
- 佩琪风配色：粉色 #F9ADC7、天蓝 #7EC8E3、草绿 #8BC34A

## 课程规划（RAZ 递进式，词汇不重复）

### 设计原则
- 每天 5 个新词，词汇不重复
- 句式从简到难递进
- 每 5 天一个复习日，混合前 4 天词汇
- 旧词在故事线中自然复现，不专门重教
- 主题持续推进，不循环

### 自然复现规则
- 每课故事线可引用之前学过的词（如 Day14 去 zoo 时复现 Day2 的动物）
- 复习桥接页用前一天 2-3 个词快速回顾
- 复习日打乱所有词，不按主题分组

## SVG 物品设计原则
- 160x160 viewBox，圆润简洁
- 主体颜色 = 教学颜色（红苹果教 red，蓝汽车教 blue）
- 2-4个基本形状组合，3岁能认出
- 可加简单表情增加可爱度
